## Запуск

1. Встановіть залежності:
   ```bash
   npm install
   ```
2. Режим розробки:
   ```bash
   npm run dev
   ```
3. Збірка продакшн:
   ```bash
   npm run build
   npm run preview
   ```

## Технології

React + Vite, Service Worker, Web App Manifest, LocalStorage.
